# App tareas

Conjunto de tareas donde se pueden gestionar diferentes tipos

## Componentes

- Componente agregar: formulario para poder agregar tareas: campturas de datos [(ngmodel)]
- Componente listar: listara mediante cartas un conjunto de tareas: ng for
- Componente buscar: listara mediante cartas un conjunto de tareas desde un buscador: ng for, ng if, ng switch

## Modelos

conjunto de clases que se trabajaran dentro de la app

- Tarea: titulo, resposonsable, fecha, prioridad (alta, media, baja)
